# exam-spotify-clone

Spotify web app based on a Figma mockup.

#  Project Goal
Demonstrate skills in semantic layout, working with the Sass (SCSS) preprocessor, and creating a responsive interface.

# Technologies
HTML5 (Semantic Markup)
CSS3 / SCSS (BEM Methodology)
Flexbox & CSS Grid (for building grids)
Google Fonts & SVG Icons

# Implemented functionality
- Complete layout of the main page.
- Custom scrollbar in Spotify style.
- Interactive elements (hover effects on cards and buttons).

 # I created the main files: index.html, style.css, and style.scss, as well as a "scss" folder where I placed all the components. I created the main layout in index.html and divided it into sections. For styling, I created corresponding scss files.

# File organization:

scss/:

_main-content.scss
_footer.scss,
...and other files for individual components.

# Architecture:
I implemented a clear separation of concerns:
Layout: Defined in index.html using HTML5 semantic tags (<header>, <aside>, <main>, <footer>).

# Styling:
 Each section has its own SCSS file in the scss directory.
